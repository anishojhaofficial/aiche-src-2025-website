## AIChE India Student Regional Conference 2025 Website

Official conference website developed for the **AIChE India Student Regional Conference 2025** hosted at the **Institute of Chemical Technology Mumbai – IOCB Campus, Bhubaneswar, India**.

The platform was independently designed, developed, deployed, and maintained to support conference operations, participant communication, and event information management.

---

### Features

- Complete frontend and backend website development
- Responsive user interface design
- Participant information and event communication workflows
- Conference schedule and event management integration
- Institutional and participant-facing deployment infrastructure

---

### Development Objectives

- Create a centralized digital platform for conference operations
- Enable streamlined participant access to conference information
- Support event coordination and communication workflows
- Develop a scalable and responsive conference web infrastructure

---

### Technologies and Infrastructure

- Frontend web development
- Backend integration and deployment
- Responsive interface architecture
- Web hosting and deployment workflows

### Project Context

The AIChE India Student Regional Conference 2025 was hosted from **21–24 August 2025** at ICT Mumbai – IOCB Campus, Bhubaneswar, bringing together participants, researchers, students, and professionals from across the region.

This repository serves as a technical showcase of the conference website infrastructure and deployment.
